SQL이 있는 문 사용

SQL Server용 Microsoft JDBC Driver 및 인라인 SQL 문을 사용하여 SQL Server 데이터베이스의 데이터에 대한 작업을 수행하는 경우 다양한 클래스를 사용할 수 있습니다. 어떤 클래스를 사용할지는 실행하려는 SQL 문의 형식에 따라 달라집니다.

SQL 문에 입력 매개 변수가 없는 경우 SQLServerStatement 클래스를 사용하고, 입력 매개 변수가 있는 경우 SQLServerPreparedStatement 클래스를 사용합니다.

입력 및 출력 매개 변수가 모두 있는 SQL 문을 사용해야 하는 경우 저장 프로시저로 구현하고 SQLServerCallableStatement 클래스를 사용하여 호출해야 합니다. 저장 프로시저 사용 방법은 저장 프로시저가 있는 문 사용을 참조하십시오.

다음 섹션에서는 SQL 문을 사용하여 SQL Server 데이터베이스 데이터 작업을 수행하는 여러 시나리오에 대해 설명합니다.
