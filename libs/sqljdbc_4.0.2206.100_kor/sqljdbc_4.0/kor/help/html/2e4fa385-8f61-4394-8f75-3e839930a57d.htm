createStatement 메서드(int, int, int)

지정된 형식, 동시성 및 유지 기능을 사용하여 SQLServerResultSet 개체를 생성하는 SQLServerStatement 개체를 만듭니다.

구문

public java.sql.Statement createStatement(int nType,
                                          int nConcur,
                                          int nHold)

매개 변수

resultSetType

결과 집합 유형을 나타내는 int 값입니다.

nConcur

결과 집합 동시성 유형을 나타내는 int 값입니다.

nHold

유지 기능을 나타내는 int 값입니다.

반환 값

Statement 개체입니다.

예외

SQLServerException

주의

이 createStatement 메서드는 java.sql.Connection 인터페이스의 createStatement 메서드에 의해 지정됩니다.
