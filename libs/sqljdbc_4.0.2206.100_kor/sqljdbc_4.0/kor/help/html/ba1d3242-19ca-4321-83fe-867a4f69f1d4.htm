ISQLServerDataSource 인터페이스

이 개체가 나타내는 데이터 원본에 대한 연결을 만들기 위한 팩터리입니다. 이 인터페이스는 SQL Server JDBC 드라이버 3.0에서 추가되었습니다.

패키지: com.microsoft.sqlserver.jdbc

확장: java.sql.CommonDataSource

구문

public interface ISQLServerDataSource

주의

이 인터페이스는 SQLServerDataSource 클래스를 통해 구현됩니다.

이 인터페이스는 다음 SQL Server용 Microsoft JDBC Driver 관련 메서드를 노출합니다.
