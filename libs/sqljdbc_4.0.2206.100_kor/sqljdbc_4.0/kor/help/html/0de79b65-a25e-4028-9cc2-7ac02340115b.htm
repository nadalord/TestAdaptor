getUnicodeStream 메서드(int)

이 SQLServerResultSet 개체의 현재 행에서 지정된 열 인덱스의 값을 검색하여 유니코드 문자의 스트림으로 반환합니다.

이 메서드는 JDBC 사양에서 더 이상 사용되지 않으며 이 메서드를 호출하면 "구현되지 않음" 예외가 발생합니다. 대신 getCharacterStream 메서드를 사용해야 합니다.

구문

public java.io.InputStream getUnicodeStream(int columnIndex)

매개 변수

columnIndex

열 인덱스를 나타내는 int입니다.

반환 값

InputStream 개체입니다.

예외

SQLServerException

주의

이 getUnicodeString 메서드는 java.sql.ResultSet 인터페이스의 getUnicodeString 메서드에 의해 지정됩니다.
