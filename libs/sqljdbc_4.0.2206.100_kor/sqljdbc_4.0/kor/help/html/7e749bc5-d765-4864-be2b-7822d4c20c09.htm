SQLServerDataSource 멤버

다음 표에는 SQLServerDataSource 클래스에 의해 노출되는 멤버가 나와 있습니다.

생성자

이름

설명

SQLServerDataSource()

SQLServerDataSource 클래스의 새 인스턴스를 초기화합니다.

필드

없음

상속된 필드

없음

메서드

이름

설명

getApplicationIntent

applicationIntent 연결 속성의 값을 반환합니다.

getApplicationName

응용 프로그램 이름을 반환합니다.

getConnection

이 SQLServerDataSource 개체가 나타내는 데이터 원본과의 연결을 설정합니다.

getDatabaseName

데이터베이스 이름을 반환합니다.

getEncrypt

encrypt 속성이 사용되는지 여부를 나타내는 Boolean 값을 반환합니다.

getDescription

데이터 원본에 대한 설명을 반환합니다.

getFailoverPartner

데이터베이스 미러링 구성에 사용되는 장애 조치(Failover) 서버의 이름을 반환합니다.

getHostNameInCertificate

SQL Server SSL(Secure Sockets Layer) 인증서의 유효성을 검사하는 데 사용되는 호스트 이름을 반환합니다.

getInstanceName

SQL Server 인스턴스 이름을 반환합니다.

getLastUpdateCount

lastUpdateCount 속성이 사용되는지 여부를 나타내는 boolean 값을 반환합니다.

getLockTimeout

데이터베이스에서 잠금 시간 초과가 보고되기 전까지 대기하는 시간(밀리초)을 나타내는 int 값을 반환합니다.

getLoginTimeout

이 SQLServerDataSource 개체가 연결을 시도하는 동안 대기하는 시간(초)을 반환합니다.

getLogWriter

모든 로깅 및 추적 메시지에 사용할 문자 출력 스트림을 반환합니다.

getMultiSubnetFailover

multiSubnetFailover 연결 속성의 값을 반환합니다.

getPacketSize

SQL Server와의 통신에 사용되는 현재 네트워크 패킷 크기(바이트)를 반환합니다.

getPortNumber

SQL Server와의 통신에 사용되는 현재 포트 번호를 반환합니다.

getReference

이 SQLServerDataSource 개체에 대한 참조를 반환합니다.

getResponseBuffering

이 SQLServerDataSource 개체에 대한 응답 버퍼링 모드를 반환합니다.

getSelectMethod

이 SQLServerDataSource 개체를 사용하여 만들어진 모든 결과 집합에 사용되는 기본 커서 유형을 반환합니다.

getSendStringParametersAsUnicode

문자열 매개 변수를 서버에 유니코드 형식으로 보낼 수 있는지 여부를 나타내는 Boolean 값을 반환합니다.

getSendTimeAsDatetime

SendTimeAsDatetime 연결 속성의 설정을 반환합니다.

getServerName

SQL Server를 실행하고 있는 컴퓨터의 이름을 반환합니다.

getTrustServerCertificate

trustServerCertificate 속성이 사용되는지 여부를 나타내는 Boolean 값을 반환합니다.

getTrustStore

인증서 trustStore 파일의 경로(파일 이름 포함)를 반환합니다.

getURL

데이터 원본에 연결하는 데 사용되는 URL을 반환합니다.

getUser

데이터 원본에 연결하는 데 사용되는 사용자 이름을 반환합니다.

getUseSQLServerBaseDate

useSQLServerBaseDate 연결 속성의 설정을 반환합니다.

getWorkstationID

데이터 원본에 연결하는 데 사용되는 클라이언트 컴퓨터 이름을 반환합니다.

getXopenStates

SQL 상태를 XOPEN 규격 상태로 변환할 수 있는지 여부를 나타내는 Boolean 값을 반환합니다.

isWrapperFor

이 데이터 원본 개체가 지정된 인터페이스에 대한 래퍼인지 여부를 나타냅니다.

setApplicationIntent

applicationIntent 연결 속성의 값을 설정합니다.

setApplicationName

응용 프로그램 이름을 설정합니다.

setAuthenticationScheme

응용 프로그램에서 사용하려는 통합 보안의 종류를 나타냅니다.

setDatabaseName

연결할 데이터베이스 이름을 설정합니다.

setDescription

데이터 원본에 대한 설명을 설정합니다.

setEncrypt

encrypt 속성이 사용되는지 여부를 나타내는 Boolean 값을 설정합니다.

setFailoverPartner

데이터베이스 미러링 구성에 사용되는 장애 조치(Failover) 서버의 이름을 설정합니다.

setHostNameInCertificate

SQL Server SSL(Secure Sockets Layer) 인증서의 유효성을 검사하는 데 사용할 호스트 이름을 설정합니다.

setInstanceName

SQL Server 인스턴스 이름을 설정합니다.

setIntegratedSecurity

integratedSecurity 속성이 사용되는지 여부를 나타내는 Boolean 값을 설정합니다.

setLastUpdateCount

lastUpdateCount 속성이 사용되는지 여부를 나타내는 Boolean 값을 설정합니다.

setLockTimeout

데이터베이스에서 잠금 시간 초과가 보고되기 전까지 대기하는 시간(밀리초)을 나타내는 int 값을 설정합니다.

setLoginTimeout

이 SQLServerDataSource 개체가 연결을 시도하는 동안 대기하는 시간(초)을 설정합니다.

setLogWriter

모든 로깅 및 추적 메시지에 사용할 문자 출력 스트림을 설정합니다.

setMultiSubnetFailover

multiSubnetFailover 연결 속성의 값을 설정합니다.

setPacketSize

SQL Server와의 통신에 사용되는 현재 네트워크 패킷 크기(바이트)를 설정합니다.

setPassword

SQL Server에 연결하는 데 사용되는 암호를 설정합니다.

setPortNumber

SQL Server와의 통신에 사용되는 포트 번호를 설정합니다.

setResponseBuffering

이 SQLServerDataSource 개체를 사용하여 만들어진 연결에 대한 응답 버퍼링 모드를 설정합니다.

setSelectMethod

이 SQLServerDataSource 개체를 사용하여 만들어진 모든 결과 집합에 사용되는 기본 커서 유형을 설정합니다.

setSendStringParametersAsUnicode

문자열 매개 변수를 서버에 유니코드 형식으로 보낼 수 있는지 여부를 나타내는 Boolean 값을 설정합니다.

setSendTimeAsDatetime

java.sql.Time 값을 서버에 보내는 방식을 지정합니다.

setServerName

SQL Server를 실행하고 있는 컴퓨터의 이름을 설정합니다.

setTrustServerCertificate

trustServerCertificate 속성이 사용되는지 여부를 나타내는 Boolean 값을 설정합니다.

setTrustStore

인증서 trustStore 파일의 경로(파일 이름 포함)를 설정합니다.

setTrustStorePassword

trustStore 데이터의 무결성을 검사하는 데 사용되는 암호를 설정합니다.

setURL

데이터 원본에 연결하는 데 사용되는 URL을 설정합니다.

setUser

데이터 원본에 연결하는 데 사용되는 사용자 이름을 설정합니다.

setWorkstationID

데이터 원본에 연결하는 데 사용되는 클라이언트 컴퓨터의 이름을 설정합니다.

setXopenStates

SQL 상태를 XOPEN 규격 상태로 변환할 수 있는지 여부를 나타내는 Boolean 값을 설정합니다.

unwrap

SQL Server용 Microsoft JDBC Driver 관련 메서드에 액세스할 수 있도록 지정된 인터페이스를 구현하는 개체를 반환합니다.
