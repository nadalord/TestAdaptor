updateDateTimeOffset(SQLServerResultSet)

이 메서드는 Microsoft SQL Server JDBC 드라이버 3.0에서 추가되었습니다.

지정된 열의 값을 DateTimeOffset 클래스 값으로 업데이트합니다.
