getSendTimeAsDatetime 메서드(SQLServerDataSource)

이 메서드는 SQL Server JDBC 드라이버 3.0에서 추가되었습니다.

sendTimeAsDatetime 연결 속성의 설정을 반환합니다.

구문

public boolean getSendTimeAsDatetime();

반환 값

java.sql.Time 값을 서버에 SQL Serverdatetime 형식으로 보내면 true이고, java.sql.Time 값을 서버에 SQL Servertime 형식으로 보내면 false입니다.

주의

sendTimeAsDatetime 연결 속성에 대한 자세한 내용은 연결 속성 설정을 참조하십시오.

SQLServerDataSource.setSendTimeAsDatetime을 사용하면 sendTimeAsDatetime 연결 속성을 프로그래밍 방식으로 설정할 수 있습니다.

자세한 내용은 java.sql.Time 값을 서버에 보내는 방식 구성을 참조하십시오.
