valueOf 메서드(java.sql.Timestamp, java.util.Calendar)

지정된 java.sql.Timestamp 값과 오프셋을 나타내는 java.util.Calendar 값을 사용하여 GMT를 기준으로 특정 오프셋의 시점을 나타내는 DateTimeOffset 개체를 만듭니다.

구문

public static DateTimeOffset valueOf(java.sql.Timestamp timestamp, java.util.Calendar calendar)

매개 변수

timestamp

java.sql.Timestamp 값입니다.

calendar

오프셋 값입니다. calendar의 날짜 및 시간 구성 요소는 timestamp 값에 따라 설정됩니다.

반환 값

지정된 java.util.Calendar 개체의 표준 시간대에서 java.sql.Timestamp 개체가 제공하는 시점을 나타내는 DateTimeOffset 개체를 반환합니다.

주의

또한 이 메서드는 java.util.Calendar 개체를 java.sql.Timestamp 개체가 제공하는 시점으로 설정합니다.
