주의

SQLServerConnectionPoolDataSource는 일반적으로 JDBC 3.0 API 사양 연결 풀링을 제공하는 Java Platform, Enterprise Edition(Java EE) 응용 프로그램 서버와 같이 기본 제공 연결 풀링을 지원하며 실제 연결을 제공하기 위해 ConnectionPoolDataSource를 필요로 하는 Java 응용 프로그램 서버 환경에서 사용됩니다.
