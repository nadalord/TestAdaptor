JDBC 드라이버 데이터 형식 이해

SQL Server용 Microsoft JDBC Driver는 SQL Server를 데이터베이스로 사용하는 Java 응용 프로그램 내의 JDBC 기본 및 고급 데이터 형식 사용을 지원합니다.

JDBC 유형 시스템은 SQL Server 데이터 형식과 Java 언어 형식 및 개체 사이의 변환을 중재합니다. JDBC 형식은 SQL-92 및 SQL-99 형식을 기준으로 합니다. JDBC 드라이버는 JDBC 사양에 따르며 예측 가능성과 융통성 사이의 올바른 균형을 제공하도록 디자인되었습니다.

이 섹션의 항목에서는 기본 및 고급 데이터 형식을 사용하는 방법 및 데이터 형식을 다른 데이터 형식으로 변환하는 방법을 설명합니다.
