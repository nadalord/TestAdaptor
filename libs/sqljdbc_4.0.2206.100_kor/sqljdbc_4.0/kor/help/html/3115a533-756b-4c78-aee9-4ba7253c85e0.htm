SQLServerConnection 멤버

다음 표에는 SQLServerConnection 클래스에 의해 노출되는 멤버가 나와 있습니다.

생성자

없음

필드

이름

설명

TRANSACTION_SNAPSHOT

스냅숏 트랜잭션 격리 수준을 지정하는 데 사용됩니다.

상속된 필드

상속하는 원본 클래스

설명

java.sql.Connection

TRANSACTION_NONE, TRANSACTION_READ_COMMITTED, TRANSACTION_READ_UNCOMMITTED, TRANSACTION_REPEATABLE_READ, TRANSACTION_SERIALIZABLE

메서드

이름

설명

clearWarnings

이 SQLServerConnection 개체에 대해 보고된 모든 경고를 지웁니다.

close

이 SQLServerConnection 개체의 데이터베이스와 JDBC 리소스를 자동으로 해제될 때까지 기다리지 않고 즉시 해제합니다.

commit

이전의 커밋 또는 롤백 후에 변경된 모든 내용을 영구적으로 만들고, 이 SQLServerConnection 개체에서 현재 보유 중인 데이터베이스 잠금을 해제합니다.

createBlob

데이터가 없는 java.sql.Blob 개체를 만듭니다.

createClob

데이터가 없는 java.sql.Clob 개체를 만듭니다.

createNClob

데이터가 없는 java.sql.NClob 개체를 만듭니다.

createStatement

데이터베이스로 SQL 문을 보내기 위한 SQLServerStatement 개체를 만듭니다.

createSQLXML

데이터가 없는 java.sql.SQLXML 개체를 만듭니다.

getAutoCommit

이 SQLServerConnection 개체에 대한 현재 자동 커밋 모드를 검색합니다.

getCatalog

이 SQLServerConnection 개체에 대한 현재 카탈로그 이름을 검색합니다.

getClientConnectionID 메서드(SQLServerConnection)

연결 시도가 성공 또는 실패했는지 여부에 관계 없이 최근 연결 시도의 연결 ID를 가져옵니다.

getClientInfo

JDBC 드라이버에서 지원되는 클라이언트 정보 속성에 대한 정보를 검색합니다.

getHoldability

이 SQLServerConnection 개체를 사용하여 만든 SQLServerResultSet 개체의 현재 유지 기능을 검색합니다.

getMetaData

이 SQLServerConnection 개체가 나타내는 연결의 대상 데이터베이스에 대한 메타데이터가 들어 있는 SQLServerDatabaseMetaData 개체를 검색합니다.

getTransactionIsolation

이 SQLServerConnection 개체에 대한 현재 트랜잭션 격리 수준을 검색합니다.

getTypeMap

이 SQLServerConnection 개체와 연결된 Map 개체를 검색합니다.

getWarnings

이 SQLServerConnection 개체에 대한 호출에서 보고된 첫 번째 경고를 검색합니다.

isClosed

이 SQLServerConnection 개체가 닫혔는지 여부를 나타냅니다.

isReadOnly

이 SQLServerConnection 개체가 읽기 전용 모드에 있는지 여부를 나타냅니다.

isValid

이 SQLServerConnection 개체가 닫히지 않고 아직 유효한지 여부를 나타냅니다.

nativeSQL

지정된 SQL 문을 데이터베이스 서버의 네이티브 SQL 문법으로 변환합니다.

prepareCall

데이터베이스 저장 프로시저를 호출하기 위한 SQLServerCallableStatement 개체를 만듭니다.

prepareStatement

데이터베이스로 매개 변수가 있는 SQL 문을 보내기 위한 SQLServerPreparedStatement 개체를 만듭니다.

releaseSavepoint

지정된 SQLServerSavepoint 개체를 현재 트랜잭션에서 제거합니다.

rollback

현재 트랜잭션에서 변경한 모든 내용을 취소하고 이 SQLServerConnection 개체가 현재 보유하고 있는 데이터베이스 잠금을 해제합니다.

setAutoCommit

이 SQLServerConnection 개체에 대한 자동 커밋 모드를 지정된 상태로 설정합니다.

setCatalog

지정된 카탈로그 이름을 설정하여 이 SQLServerConnection 개체의 데이터베이스 중 작업할 하위 공간을 선택합니다.

setClientInfo

클라이언트 정보 속성의 값을 설정합니다.

setHoldability

이 SQLServerSavepoint 개체를 사용하여 만든 SQLServerResultSet 개체의 유지 기능을 변경합니다.

setReadOnly

JDBC 드라이버에서 데이터베이스 최적화를 수행하기 위한 힌트로 사용할 수 있도록 이 SQLServerConnection 개체를 읽기 전용 모드로 설정합니다.

setSavepoint

현재 트랜잭션에 명명되지 않은 저장점을 만들고 이 저장점을 나타내는 새 SQLServerSavepoint 개체를 반환합니다.

setTransactionIsolation

이 SQLServerConnection 개체에 대한 트랜잭션 격리 수준을 지정된 수준으로 변경합니다.

setTypeMap

지정된 TypeMap 개체를 이 SQLServerConnection 개체에 대한 형식 맵으로 설치합니다.
