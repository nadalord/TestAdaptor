설명

addBatch

SQLServerPreparedStatement에서 상속되며, 이 CallableStatement 개체의 명령 일괄 처리에 매개 변수 집합을 추가합니다.

cancel

SQLServerStatement에서 상속되며, 이 CallableStatement 개체에 의해 현재 실행되고 있는 SQL 문을 취소합니다.

clearBatch

SQLServerPreparedStatement에서 상속되며, 이 CallableStatement 개체에 대한 SQL 명령의 현재 목록을 비웁니다.

clearParameters

SQLServerPreparedStatement에서 상속되며, 현재 매개 변수 값을 즉시 지웁니다.

clearWarnings

SQLServerStatement에서 상속되며, 이 CallableStatement 개체에 대해 보고된 모든 경고를 지웁니다.

close

SQLServerPreparedStatement에서 상속되며, 이 CallableStatement 개체의 데이터베이스와 JDBC 리소스를 자동으로 해제될 때까지 기다리지 않고 즉시 해제합니다.

execute

SQLServerPreparedStatement에서 상속되며, 이 CallableStatement 개체에서 모든 종류의 SQL 문을 실행합니다.

executeBatch

SQLServerPreparedStatement에서 상속되며, 실행할 명령 일괄 처리를 데이터베이스로 전송합니다. 모든 명령이 성공적으로 실행되면 업데이트 횟수의 배열을 반환합니다.

executeQuery

SQLServerPreparedStatement에서 상속되며, 이 CallableStatement 개체에서 SQL 쿼리를 실행하고 쿼리에 의해 생성된 SQLServerResultSet 개체를 반환합니다.

executeUpdate

SQLServerPreparedStatement에서 상속되며, 이 CallableStatement 개체에서 SQL INSERT, UPDATE, MERGE 또는 DELETE 문과 같은 SQL 문이나 DDL 문 같이 아무 것도 반환하지 않는 SQL 문을 실행합니다.

getConnection

SQLServerStatement에서 상속되며, 이 CallableStatement 개체를 생성한 SQLServerConnection 개체를 검색합니다.

getDateTimeOffset

지정된 열의 값을 검색하여 DateTimeOffset 클래스 개체로 반환합니다.

getFetchDirection

SQLServerStatement에서 상속되며, 이 CallableStatement 개체에서 생성된 결과 집합에 기본값으로 사용되는 데이터베이스 테이블에서의 행 인출 방향을 검색합니다.

getFetchSize

SQLServerStatement에서 상속되며, 이 CallableStatement 개체에서 생성된 결과 집합 개체의 기본 인출 크기로 사용되는 결과 집합 행 수를 검색합니다.

getGeneratedKeys

SQLServerStatement에서 상속되며, 이 CallableStatement 개체를 실행한 결과로 만들어진 자동 생성 키를 검색합니다.

getMaxFieldSize

SQLServerStatement에서 상속되며, 이 CallableStatement 개체에 의해 생성된 SQLServerResultSet 개체의 문자 및 이진 열 값에 대해 반환될 수 있는 최대 바이트 수를 검색합니다.

getMaxRows

SQLServerStatement에서 상속되며, 이 CallableStatement 개체에 의해 생성된 SQLServerResultSet 개체에 포함될 수 있는 최대 행 수를 검색합니다.

getMetaData

SQLServerPreparedStatement에서 상속되며, 이 CallableStatement 개체가 실행될 때 반환되는 SQLServerResultSet 개체의 열에 대한 정보가 들어 있는 SQLServerResultSetMetaData 클래스 개체를 검색합니다.

getMoreResults

SQLServerStatement에서 상속되며, 이 CallableStatement 개체의 다음 결과로 이동합니다.

getParameterMetaData

SQLServerPreparedStatement에서 상속되며, 이 CallableStatement 개체에 대한 매개 변수의 수, 형식 및 속성을 검색합니다.

getArray

지정된 매개 변수의 값을 검색하여 Array 개체로 반환합니다.

getBlob

getboolean
