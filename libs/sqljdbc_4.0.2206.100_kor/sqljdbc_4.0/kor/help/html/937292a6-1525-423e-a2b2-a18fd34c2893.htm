SQLServerConnection 클래스

MicrosoftSQL Server 데이터베이스에 대한 JDBC 연결을 나타냅니다.

패키지: com.microsoft.sqlserver.jdbc

구현:ISQLServerConnection, java.io.Serializable

구문

public class SQLServerConnection

주의

SQLServerConnection은 JDBC 연결 풀링을 지원하며, 실제 JDBC 연결이거나 논리적 JDBC 연결일 수 있습니다. SQLServerConnection은 해당 연결에서 만들어진 모든 문에 대한 트랜잭션 제어를 관리하며, XAResource 어댑터를 통해 관리되는 XA 분산 트랜잭션에 참여할 수 있습니다.

SQLServerConnection은 PreparedStatement 핸들의 풀을 관리합니다. PreparedStatement는 한 번만 준비하면 매개 변수의 데이터 값을 달리 하여 여러 번 실행할 수 있는 문입니다. PreparedStatement는 논리적(풀링된) 연결 닫기를 통해서도 유지 관리됩니다.

SQLServerConnection은 스레드로부터 안전하지 않습니다. 그러나 단일 연결에서 만들어진 여러 문을 현재 스레드에서 동시에 처리할 수 있습니다.

이 클래스는 SQLServerConnection 클래스, java.sql.connection 인터페이스 및 ISQLServerConnection 인터페이스에 대한 래핑 해제를 지원합니다. 자세한 내용은 래퍼 및 인터페이스를 참조하십시오.
