설명

addBatch

이 Statement 개체의 명령 일괄 처리에 매개 변수 집합을 추가합니다.

cancel

SQLServerStatement에서 상속되며, 이 Statement 개체에 의해 현재 실행되고 있는 SQL 문을 취소합니다.

clearBatch

이 SQLServerStatement 개체에 대한 SQL 명령의 현재 목록을 비웁니다.

clearParameters

현재 매개 변수 값을 즉시 지웁니다.

clearWarnings

SQLServerStatement에서 상속되며, 이 Statement 개체에 대해 보고된 모든 경고를 지웁니다.

close

이 Statement 개체의 데이터베이스와 JDBC 리소스를 자동으로 해제될 때까지 기다리지 않고 즉시 해제합니다.

execute

이 Statement 개체에서 모든 종류의 SQL 문을 실행합니다.

executeBatch

실행할 명령 일괄 처리를 데이터베이스로 전송합니다. 모든 명령이 성공적으로 실행되면 업데이트 횟수의 배열을 반환합니다.

executeQuery

이 Statement 개체에서 SQL 쿼리를 실행하고 쿼리에 의해 생성된 SQLServerResultSet 개체를 반환합니다.

executeUpdate

이 Statement 개체에서 SQL INSERT, UPDATE, MERGE 또는 DELETE 문과 같은 SQL 문이나 DDL 문 같이 아무 것도 반환하지 않는 SQL 문을 실행합니다.

getConnection

SQLServerStatement에서 상속되며, 이 Statement 개체를 생성한 SQLServerConnection 개체를 검색합니다.

getFetchDirection

SQLServerStatement에서 상속되며, 이 Statement 개체에서 생성된 결과 집합에 기본값으로 사용되는 데이터베이스 테이블에서의 행 인출 방향을 검색합니다.

getFetchSize

SQLServerStatement에서 상속되며, 이 Statement 개체에서 생성된 결과 집합 개체의 기본 인출 크기로 사용되는 결과 집합 행 수를 검색합니다.

getGeneratedKeys

SQLServerStatement에서 상속되며, 이 Statement 개체를 실행한 결과로 만들어진 자동 생성 키를 검색합니다.

getMaxFieldSize

SQLServerStatement에서 상속되며, 이 Statement 개체에 의해 생성된 SQLServerResultSet 개체의 문자 및 이진 열 값에 대해 반환될 수 있는 최대 바이트 수를 검색합니다.

getMaxRows

SQLServerStatement에서 상속되며, 이 Statement 개체에 의해 생성된 SQLServerResultSet 개체에 포함될 수 있는 최대 행 수를 검색합니다.

getMetaData

이 Statement 개체가 실행될 때 반환되는 SQLServerResultSet 개체의 열에 대한 정보가 들어 있는 SQLServerResultSetMetaData 클래스 개체를 검색합니다.

getMoreResults

SQLServerStatement에서 상속되며, 이 Statement 개체의 다음 결과로 이동합니다.

getParameterMetaData

이 Statement 개체에 대한 매개 변수의 수, 형식 및 속성을 검색합니다.

getResponseBuffering

SQLServerStatement에서 상속되며, 이 SQLServerStatement 개체에 대한 응답 버퍼링 모드를 검색합니다.

getQueryTimeout

SQLServerStatement에서 상속되며, SQL Server용 Microsoft JDBC Driver에서 이 Statement 개체가 실행될 때까지 대기하는 시간(초)을 검색합니다.

getResultSet

SQLServerStatement에서 상속되며, 현재 결과를 검색하여 SQLServerResultSet 개체로 반환합니다.

getResultSetConcurrency

SQLServerStatement에서 상속되며, 이 Statement 개체에 의해 생성된 SQLServerResultSet 개체의 결과 집합 동시성을 검색합니다.

getResultSetHoldability

SQLServerStatement에서 상속되며, 이 Statement 개체에 의해 생성된 SQLServerResultSet 개체의 결과 집합 유지 기능을 검색합니다.

getResultSetType

SQLServerStatement에서 상속되며, 이 Statement 개체에 의해 생성된 SQLServerResultSet 개체의 결과 집합 유형을 검색합니다.

getUpdateCount

SQLServerStatement에서 상속되며, 현재 결과를 검색하여 업데이트 횟수로 반환합니다.

getWarnings

SQLServerStatement에서 상속되며, 이 Statement 개체에 대한 호출에서 보고된 첫 번째 경고를 검색합니다.

isClosed

SQLServerStatement에서 상속되며, 이 Statement 개체가 닫혔는지 여부를 나타냅니다.

isPoolable

SQLServerStatement에서 상속되며, 사용자가 제공한 문 풀에 문을 추가할 수 있는지 여부를 나타내는 값을 반환합니다.

isWrapperFor

이 문 개체가 지정된 인터페이스에 대한 래퍼인지 여부를 나타냅니다.

setArray

지정된 매개 변수 번호를 지정된 Array 개체로 설정합니다.

setDate

setDateTimeOffset

setDouble

setEscapeProcessing

setFetchDirection

setFetchSize

SQLServerStatement에서 상속되며, 추가 행이 필요할 경우 데이터베이스에서 인출되는 행 수에 관한 힌트를 JDBC 드라이버에 제공합니다.

setFloat

setMaxFieldSize

setMaxRows

SQLServerStatement에서 상속되며, SQLServerResultSet 개체에 포함될 수 있는 최대 행 수에 대한 제한을 지정된 수로 설정합니다.

setNCharacterStream

setNull

setNString

setQueryTimeout

SQLServerStatement에서 상속되며, 드라이버에서 Statement 개체가 실행될 때까지 대기하는 시간(초)을 지정된 시간(초)으로 설정합니다.

setRef

setShort

setUnicodeStream

setURL
