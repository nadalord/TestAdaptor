SQLServerStatement 멤버

다음 표에는 SQLServerStatement 클래스에 의해 노출되는 멤버가 나와 있습니다.

생성자

없음

필드

없음

상속된 필드

이름

설명

java.sql.Statement

CLOSE_ALL_RESULTS, CLOSE_CURRENT_RESULT, EXECUTE_FAILED, KEEP_CURRENT_RESULT, NO_GENERATED_KEYS, RETURN_GENERATED_KEYS, SUCCESS_NO_INFO

메서드

이름

설명

addBatch

이 SQLServerStatement 개체에 대한 명령의 현재 목록에 지정된 SQL 명령을 추가합니다.

cancel

이 SQLServerStatement 개체에 의해 현재 실행되고 있는 SQL 문을 취소합니다.

clearBatch

이 SQLServerStatement 개체에 대한 SQL 명령의 현재 목록을 비웁니다.

clearWarnings

이 SQLServerStatement 개체에 대해 보고된 모든 경고를 지웁니다.

close

이 SQLServerStatement 개체의 데이터베이스와 JDBC 리소스를 자동으로 해제될 때까지 기다리지 않고 즉시 해제합니다.

execute

여러 결과를 반환할 수 있는 지정된 SQL 문을 실행합니다.

executeBatch

실행할 명령 일괄 처리를 데이터베이스로 전송합니다. 모든 명령이 성공적으로 실행되면 업데이트 횟수의 배열을 반환합니다.

executeQuery

지정된 SQL 문을 실행하고 단일 SQLServerResultSet 개체를 반환합니다.

executeUpdate

INSERT, UPDATE, MERGE 또는 DELETE 문과 같은 지정된 SQL 문이나 SQL DDL 문 같이 아무 것도 반환하지 않는 SQL 문을 실행합니다.

getConnection

이 SQLServerStatement 개체를 생성한 SQLServerConnection 개체를 검색합니다.

getFetchDirection

이 SQLServerStatement 개체에서 생성된 결과 집합에 기본값으로 사용되는 데이터베이스 테이블에서의 행 인출 방향을 검색합니다.

getFetchSize

이 SQLServerStatement 개체에서 생성된 결과 집합 개체의 기본 인출 크기로 사용되는 결과 집합 행 수를 검색합니다.

getGeneratedKeys

이 SQLServerStatement 개체를 실행한 결과로 만들어진 자동 생성 키를 검색합니다.

getMaxFieldSize

이 SQLServerStatement 개체에 의해 생성된 SQLServerResultSet 개체의 문자 및 이진 열 값에 대해 반환될 수 있는 최대 바이트 수를 검색합니다.

getMaxRows

이 SQLServerStatement 개체에 의해 생성된 SQLServerResultSet 개체에 포함될 수 있는 최대 행 수를 검색합니다.

getMoreResults

이 SQLServerStatement 개체의 다음 결과로 이동합니다.

getQueryTimeout

SQL Server용 Microsoft JDBC Driver에서 이 SQLServerStatement 개체가 실행될 때까지 대기하는 시간(초)을 검색합니다.

getResponseBuffering

이 SQLServerStatement 개체에 대한 응답 버퍼링 모드를 검색합니다.

getResultSet

현재 결과를 검색하여 SQLServerResultSet 개체로 반환합니다.

getResultSetConcurrency

이 SQLServerStatement 개체에 의해 생성된 SQLServerResultSet 개체의 결과 집합 동시성을 검색합니다.

getResultSetHoldability

이 SQLServerStatement 개체에 의해 생성된 SQLServerResultSet 개체의 결과 집합 유지 기능을 검색합니다.

getResultSetType

이 SQLServerStatement 개체에 의해 생성된 SQLServerResultSet 개체의 결과 집합 유형을 검색합니다.

getUpdateCount

현재 결과를 검색하여 업데이트 횟수로 반환합니다.

getWarnings

이 SQLServerStatement 개체에 대한 호출에서 보고된 첫 번째 경고를 검색합니다.

isClosed

이 SQLServerStatement 개체가 닫혔는지 여부를 나타냅니다.

isPoolable

사용자가 제공한 문 풀에 문을 추가할 수 있는지 여부를 나타내는 값을 반환합니다.

isWrapperFor

이 문 개체가 지정된 인터페이스에 대한 래퍼인지 여부를 나타냅니다.

setCursorName

SQL 커서 이름을 이후 실행 메서드에 사용될 지정된 문자열로 설정합니다.

setEscapeProcessing

이스케이프 처리 모드를 설정합니다.

setFetchDirection

결과 집합 행을 처리할 방향에 관한 힌트를 JDBC 드라이버에 제공합니다.

setFetchSize

추가 행이 필요할 경우 데이터베이스에서 인출되는 행 수에 관한 힌트를 JDBC 드라이버에 제공합니다.

setMaxFieldSize

문자 또는 이진 값을 저장하는 SQLServerResultSet 열의 최대 바이트 수에 대한 제한을 지정된 바이트 수로 설정합니다.

setMaxRows

SQLServerResultSet 개체에 포함될 수 있는 최대 행 수에 대한 제한을 지정된 수로 설정합니다.

setPoolable

문을 풀링하거나 풀링하지 않도록 요청합니다.

setQueryTimeout

드라이버에서 SQLServerStatement 개체가 실행될 때까지 대기하는 시간(초)을 지정된 시간(초)으로 설정합니다.

setResponseBuffering

이 SQLServerStatement 개체의 응답 버퍼링 모드를 대/소문자를 구분하지 않는 String full 또는 adaptive로 설정합니다.

unwrap

SQL Server용 Microsoft JDBC Driver 관련 메서드에 액세스할 수 있도록 지정된 인터페이스를 구현하는 개체를 반환합니다.
