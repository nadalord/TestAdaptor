드라이버 버전 가져오기

설치된 SQL Server용 Microsoft JDBC Driver 버전은 다음과 같은 방법으로 찾을 수 있습니다.

  • SQLServerDatabaseMetaData 메서드 getDriverMajorVersion, getDriverMinorVersion 또는 getDriverVersion을 호출합니다.

  • 버전은 제품 배포의 readme.txt 파일에 표시됩니다.

또한 JDBC 드라이버 이름은 SQLServerDatabaseMetaData 클래스의 getDriverName 메서드 호출에서 반환될 수 있습니다. 이 메서드는 "SQL Server용 Microsoft JDBC Driver 4.0"을 반환합니다.

다음은 SQLServerDatabaseMetaData 클래스의 메서드를 호출하는 경우의 출력 예입니다.

getDriverName = Microsoft JDBC Driver 4.0 for SQL Server

getDriverMajorVersion = 4

getDriverMinorVersion = 0

getDriverVersion = 4.0.xxx.x

여기서 "xxx.x"는 최종 버전 번호입니다.
