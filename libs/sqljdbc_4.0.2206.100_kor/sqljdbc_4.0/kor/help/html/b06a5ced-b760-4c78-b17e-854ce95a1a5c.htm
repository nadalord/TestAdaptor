getClientInfo 메서드()

JDBC 드라이버에서 지원되는 각 클라이언트 정보 속성의 이름 및 현재 값이 들어 있는 목록을 검색합니다.

구문

public java.util.Properties getClientInfo()

반환 값

드라이버에서 지원되는 각 클라이언트 정보 속성의 이름 및 현재 값이 들어 있는 Properties 개체입니다.

예외

SQLServerException

주의

이 getClientInfo 메서드는 java.sql.Connection 인터페이스의 getClientInfo 메서드에 의해 지정됩니다.

SQL Server용 Microsoft JDBC Driver에서는 클라이언트 정보 속성을 지원하지 않습니다. 결과적으로 이 메서드는 빈 Properties 개체를 반환합니다.

마찬가지로 응용 프로그램에서는 SQLServerDatabaseMetaData 클래스의 getClientInfoProperties 메서드를 사용하여 드라이버에서 지원하는 클라이언트 정보 속성의 목록을 검색할 수 있습니다. getClientInfoProperties 메서드는 빈 결과 집합을 반환합니다.
