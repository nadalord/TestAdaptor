SQLServerDataSourceObjectFactory 클래스

JNDI(Java Naming and Directory Interface)의 데이터 원본을 구체화하기 위한 개체 팩터리를 나타냅니다.

패키지: com.microsoft.sqlserver.jdbc

확장: java.lang.Object

구현: javax.naming.spi.ObjectFactory

구문

public class SQLServerDataSourceObjectFactory

주의

모든 데이터 원본 클래스는 이 메서드를 상속합니다. 참조 가능한 인터페이스에 대한 지원의 일부로 SQL Server용 Microsoft JDBC Driver에서는 ObjectFactory를 구현하는 이 클래스를 노출합니다. Java 응용 프로그램 서버에서는 데이터 원본 클래스에 대해 getReference를 호출하고, 내부적으로 클래스 이름을 해당 클래스 팩터리로 사용하는 Reference 개체를 만듭니다.

Java 응용 프로그램 서버에서는 Reference 개체를 역참조해야 하는 경우 SQLServerDataSourceObjectFactory 개체의 인스턴스를 만들고 Reference 개체를 전달하여 getObjectInstance 메서드를 호출하는 방식으로 데이터 원본 인스턴스를 검색합니다.
