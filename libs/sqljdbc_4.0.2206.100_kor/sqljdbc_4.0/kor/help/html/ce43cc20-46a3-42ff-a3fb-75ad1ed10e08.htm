데이터 연결 및 검색

SQL Server용 Microsoft JDBC Driver를 사용하는 경우 두 가지 기본 방법을 사용하여 SQL Server 데이터베이스에 연결을 설정할 수 있습니다. 그 중 하나가 연결 URL에 연결 속성을 설정한 다음 DriverManager 클래스의 getConnection 메서드를 호출하여 SQLServerConnection 개체를 반환하는 것입니다.

JDBC 드라이버가 지원하는 연결 속성 목록은 연결 속성 설정을 참조하십시오.

두 번째 방법은 SQLServerDataSource 클래스의 setter 메서드를 사용하여 연결 속성을 설정한 다음 getConnection 메서드를 호출하여 SQLServerConnection 개체를 반환하는 것입니다.

이 섹션의 항목에서는 이와 같이 SQL Server 데이터베이스에 연결할 수 있는 서로 다른 방법을 설명하고, 데이터 검색에 대한 기술 차이를 보여 줍니다.
