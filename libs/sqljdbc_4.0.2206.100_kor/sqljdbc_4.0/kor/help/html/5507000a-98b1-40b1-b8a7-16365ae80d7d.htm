getMinorVersion 메서드(SQLServerDriver)

SQL Server용 Microsoft JDBC Driver의 부 버전 번호를 반환합니다.

구문

public int getMinorVersion()

반환 값

JDBC 드라이버의 부 버전 번호가 들어 있는 int입니다.

주의

이 getMinorVersion 메서드는 java.sql.Driver 인터페이스의 getMinorVersion 메서드에 의해 지정됩니다.
