JDBC 드라이버로 SQL Server에 연결

SQL Server용 Microsoft JDBC Driver로 수행하는 가장 기본적인 작업 중 하나는 SQL Server 데이터베이스에 연결하는 것입니다. 데이터베이스와의 모든 상호 작용은 SQLServerConnection 개체를 통해 이루어지며, JDBC 드라이버가 이처럼 평면 아키텍처이기 때문에 거의 모든 주요 동작에서 SQLServerConnection 개체에 영향을 미칩니다.

SQL Server가 IPv6 포트에서만 수신 대기하는 경우 java.net.preferIPv6Addresses 시스템 속성을 설정하여 SQL Server에 연결하는 데 IPv4 대신 IPv6이 사용되도록 합니다.

System.setProperty("java.net.preferIPv6Addresses", "true");
