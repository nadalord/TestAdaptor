JDBC 드라이버 개요

SQL Server용 Microsoft JDBC Driver는 Microsoft SQL Server 데이터베이스에 대한 강력한 데이터 액세스 기능을 제공하는 JDBC(Java Database Connectivity) 4.0 호환 드라이버입니다. JDBC 드라이버는 데이터베이스 미러링과 XML, 사용자 정의 및 큰 값 데이터 형식을 비롯하여 SQL Server 2005에서 도입된 다양한 기능에 액세스할 수 있으며 새로운 "스냅숏" 트랜잭션 격리를 지원합니다. 또한 JDBC 드라이버는 통합 인증을 사용할 수 있도록 지원합니다.

이 섹션의 항목에서는 JDBC 드라이버의 일반적인 개요를 포함하여 JDBC 드라이버를 사용하기 위한 시스템 요구 사항, JDBC 드라이버의 사용 방법 및 자세한 내용을 확인할 수 있는 위치를 소개합니다.
