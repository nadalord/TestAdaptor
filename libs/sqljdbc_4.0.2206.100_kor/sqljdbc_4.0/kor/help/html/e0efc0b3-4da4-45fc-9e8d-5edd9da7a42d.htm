getTime 메서드(java.lang.String)(ISQLServerResultSet)

이 SQLServerResultSet 개체의 현재 행에서 지정된 열 이름의 값을 검색하여 Java 프로그래밍 언어의 java.sql.Time 개체로 반환합니다.

구문

public java.sql.Time getTime(java.lang.String columnName)

매개 변수

columnName

열 이름이 들어 있는 String입니다.

반환 값

Time 개체입니다.

예외

SQLServerException

주의

이 getTime 메서드는 java.sql.ResultSet 인터페이스의 getTime 메서드에 의해 지정됩니다.

이 메서드는 SQL Server datetime 또는 smalldatetime 데이터 형식의 유효한 시간 부분을 반환합니다. 이때 날짜 부분은 Java 기준 날짜인 1970-01-01로 설정됩니다.
